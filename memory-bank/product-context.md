# Product Context

## Problem
Field workers and busy people need to update checklists while their hands are occupied. Traditional checklist apps require finding the right item and tapping a checkbox — interrupting workflow.

## Solution
CheckVoice lets users check off items by:
1. **Speaking** — "I just finished painting the hallway" → checks off "Paint hallway"
2. **Taking a photo** — photo of clean room → checks off "Clean bedroom"
3. **Asking questions** — "What's next?" → returns next uncompleted item

## Target Users
- Field workers (contractors, maintenance crews)
- Homeowners managing renovation/chore projects
- Anyone who wants hands-free checklist management

## UX Philosophy: Sonic Clarity
Editorial, premium aesthetic. Manrope headlines + Inter body. No borders — boundaries through background color shifts. Glassmorphism for recording HUD. Large touch targets (48dp+). Mobile-first but responsive to desktop.

## Key UX Flows
1. Dashboard → Project list → Checklist → Items (manual or AI-assisted)
2. Voice: Tap mic → speak → confirmation toast → item checked
3. Photo: Tap camera → capture/upload → item checked
4. Assistant: Chat-like interface for questions about checklist status
