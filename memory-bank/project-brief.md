# Project Brief: CheckVoice

## Scope
Mobile-first web app for managing checklists via voice, photos, and text. Users create projects with checklists, then check off items by speaking naturally, snapping photos, or asking questions.

## Goals
1. Voice-to-check: "I just washed the car" → "Wash Car" checked off
2. Photo-to-check: photo of clean room → "Clean bedroom" checked off
3. Natural language queries: "What's next?" → returns next uncompleted item
4. Editorial, premium UI following "Sonic Clarity" design system

## Requirements Summary
- 26 requirements (R-001 through R-026) — see PRESEARCH.md for full registry
- Must-have: CRUD, voice, photo, AI tool use, Sonic Clarity UI, mobile-first
- Should-have: PWA, deployment, Sonic Pulse animation

## Target Users
Field workers, contractors, anyone who needs hands-free checklist updates.
